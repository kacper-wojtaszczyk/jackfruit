# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting significant technical decisions in the Jackfruit project.

## Index

| ADR | Title | Status |
|-----|-------|--------|
| [001](001-grid-data-storage.md) | Grid Data Storage | Accepted |

## Format

Each ADR follows a lightweight format:

- **Status**: Proposed / Accepted / Superseded
- **Context**: What problem are we solving? What options exist?
- **Decision**: What did we choose and why?
- **Consequences**: What are the trade-offs? How do we mitigate downsides?
